Flexible Box

Exitem quatro tipos de configuraçôes permitidas no flexbox pai flex direction entre elas estão:

  • row (padrão) esquerda para direita
  • row-reverse direita para esquerda
  • column cima para baixo
  • column-reverse baixo para cima

Eixos principais e cruzados:

  • Eixo principal (main axis): é o eixo definido pela direção do flexbox (flex direction)
  • Eixo cruzado (cross axis): é o eixo perpendicular ao eixo principal

Eixos contem main-start e main-end, cross-start e cross-end.
